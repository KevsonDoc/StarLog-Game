## Como baixar o projeto
```bash
    #Clonar repositório
    $ git clone 
    
    #Acessar repositório
    $ cd StarLog-Game
```
---
## Instalando programas e dependência para executar a aplicação.
```bash
#Instalando dependências necessárias para aplicação
$ yarn install

    ou

$ npm install
```

---

### Iniciando aplicação

```bash
    #Inicia aplicação no http:localhost: 2000
    $ yarn dev
```
